# Manual QA: Яндекс Самокат

Портфолио-кейс по ручному тестированию сервиса «Яндекс Самокат».

Проект показывает работу с тест-анализом, тест-дизайном, чек-листами, тест-кейсами, тестовыми данными и баг-репортами.

## Что проверялось

- веб-приложение «Яндекс Самокат»;
- оформление заказа;
- статус заказа;
- валидация полей формы заказа;
- отображение данных заказа;
- кроссбраузерность и разные разрешения экрана;
- мобильное приложение курьера;
- push-уведомления и сценарии работы с заказами.

## Артефакты

| Файл | Содержание |
|---|---|
| `CHECKLIST.md` | Чек-лист проверок веб-приложения по браузерам и разрешениям |
| `TEST_DATA.md` | Тестовые данные и классы эквивалентности для валидации форм |
| `TEST_CASES.md` | Тест-кейсы для мобильного приложения курьера |
| `BUG_REPORTS.md` | Баг-репорты и дефекты, найденные вне тестовой документации |
