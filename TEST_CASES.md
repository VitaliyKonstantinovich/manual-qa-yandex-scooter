# Тест-кейсы: мобильное приложение курьера

Тест-кейсы описывают проверки мобильного приложения курьера на Android Emulator.

## ТК 01. Успешная авторизация

**Предусловия:** Приложение установлено. Открыт экран авторизации.

**Окружение:** Эмулятор Android Studio, Android 9.0

**Шаги:**

1. Ввести допустимый логин и допустимый пароль
2. Нажать кнопку «Войти»

**Ожидаемый результат:** Поля заполнены введёнными значениями. Открывается экран списка заказов. Активна вкладка «Все»
